<Tip>

**10x 빠르게 세일즈 하는 법**: Relate에는 더 빠르게 움직일 수 있도록 간단하고 기억하기 쉬운 단축키들을 지원하고 있습니다. **앞으로도 계속해서 더 많은 단축키를 지원할 예정입니다**.

</Tip>

### Inbox

| Action              | Shortcut                          |
| ------------------- | --------------------------------- |
| Archive 하기        | `E`                               |
| 아래로 가기 `↓`     | `J` or `↓`                        |
| 위로 가기`↑`        | `K` or `↑`                        |
| 댓글 등록하기       | `⌘` + `Return` (`Ctrl` + `Enter`) |

### 검색(Search) & Command Bar

Relate 앱내 어디서든 동작하는 유니버설 단축키로 검색과 다양한 액션을 할 수 있습니다.

![Command-bar](/images/features/command-bar.png)

| Action                  | Shortcut  |
| ----------------------- | --------- |
| 검색 & Command Bar 열기 | `⌘` + `K` |

Command Bar에서 할 수 있는 액션은 다음과 같습니다. 

- Compose email: 새로운 이메일 작성
- Create email template: 새로운 이메일 템플릿 생성
- Create note: 새로운 노트 작성
- Create note template: 새로운 노트 템플릿 생성
- Search organizations & people: 고객사 이름, 담당자 이름, 담당자 이메일 또는 커스텀 필드 중 Text Type의 데이터로 검색

추후 Command Bar를 통한 새로운 Organization, Contact, Deal 등록 등 다양한 기능을 지원할 예정입니다.
