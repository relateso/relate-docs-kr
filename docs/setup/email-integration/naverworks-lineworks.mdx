## 1. Naverworks IMAP/SMTP 활성화 확인하기

네이버웍스의 [설정 페이지](https://mail.worksmobile.com/settings/) (로그인 필요)로 이동하셔서 **메일 > 고급 설정**에 접근하신 다음, IMAP/SMTP 탭이 보이는지 확인해주세요. (보이지 않는다면, IMAP/SMTP가 활성화 되지 않은 것이므로 2번으로 넘어가주세요)

![Naverworks IMAP/SMTP 활성화 확인하기](/docs/images/setup/naverworks-lineworks/naverworks-imap.png)

탭을 눌러서 접근하시면 아래와 같은 정보를 확인하실 수 있습니다. **여기까지 확인하셨으면, 사전 준비가 완료되었습니다.**

![Naverworks 정보](/docs/images/setup/naverworks-lineworks/naverworks-imap-info.png)

위에 나와있는 정보를 기억해주세요.

IMAP: `imap.worksmobile.com` / IMAP 포트: 993, SSL 연결 필요

SMTP: `smtp.worksmobile.com` / SMTP 포트: 587, SSL 연결 필요

## 2. Naverworks IMAP/SMTP 활성화 하기

**팀내 네이버웍스 담당자(Admin) 만이 아래 설정을 확인하실 수 있습니다.**

Naverworks 의 [관리자 페이지](https://admin.worksmobile.com/)로 이동해 로그인 해주세요.

좌측에 있는 **보안 > 서비스 권한** 페이지로 접근하신 뒤, 권한 설정 탭 아래 있는 서비스 이용 권한 페이지로 접근해주세요.

![Naverworks Admin Portal](/docs/images/setup/naverworks-lineworks/naverworks-admin.png)

이 페이지에서 `IMAP/SMTP` 가 **사용**으로 되어 있는지 확인해주세요. 만약 사용하고 있지 않다면 이 페이지에서 우측에 있는 **권한 수정** 버튼을 눌러 `IMAP/SMTP` 에 체크하신 후, 저장하셔서 `IMAP/SMTP 사용` 이 될 수 있게 해주세요.

![Naverworks IMAP in-use](/docs/images/setup/naverworks-lineworks/imap-in-use.png)

<Note>
  💡 만일 계정에 SSO나 2FA를 사용하고 계신다면, 일시적으로 해지를 해주시고,
  Relate 연동 후에 다시 연결해주시면 됩니다.
</Note>
