## Overview

Relate에 갖고 계신 기존 고객데이터를 쉽고 간편하게 import 할 수 있는 방법을 소개합니다.

<Check>

**1-point guide**

[Admin 페이지](https://admin.relate.so/) 왼쪽 메뉴에서 Import Data를 클릭한 뒤, Import Organizations 또는 Import Deals 버튼을 클릭합니다.

![Relate Data](/images/setup/import_menu1.png)

Relate에 고객 정보를 Import 할 때는 Organization과 People은 Import Organizations에서, Deal은 Import Deals에서 할 수 있습니다.

</Check>


## 임포트 전에 알아야 할 것 (Before Importing)

### 고객 데이터 종류 (Data Types)

Relate의 고객 정보는 3가지로 나눠집니다. 
- [Organization (회사)](https://www.relate.kr/docs/features/organizations)
- [Person (회사 소속 이해관계자)](https://www.relate.kr/docs/features/people)
- [Deal (영업 거래)](https://www.relate.kr/docs/features/deal-pipeline)

![Relate Data](/images/setup/org_person_deal.png)

<Info>

Relate 내에서 고객 정보를 분류할 때 Orgnization이 가장 기본이 되며, Person과 Deal은 Organization내에 속하게 되어 있습니다.

</Info>

### 디폴트 데이터 필드 (Default Data Fields)

Organization, Person, Deal를 생성할 때 필수 입력 정보와 디폴트 데이터 필드는 다음과 같습니다.

**Organization:**
- **Organization Name (필수 입력 정보)**
- Website
- Description
- Assignee (팀 내 고객 담당자)

**Person:**
- **Person Name (필수 입력 정보)**
- Title
- Email
- Phone

**Deal:**
- **Organization (필수 입력 정보)**
- Person
- Value
- Prospect Status (잠재 고객 상태)
- Pipeline (2개 이상의 파이프라인을 사용할 경우만 해당)
- Pipeline Stage
- Assignee (팀 내 거래 담당자)
- Notes


## 임포트(Import) 하기

먼저 Relate의 [관리자 페이지](https://admin.relate.so/)로 이동한 후 왼쪽 메뉴 바에서 `Import Data`를 선택하고, 그 다음 `Import Organizations` 이나 `Import Deals`를 선택하세요.

![Relate Data](/images/setup/import_menu1.png)

### Step 1. Getting Started
가져올 수 있는 .CSV 파일이 이미 준비되어 있다면 바로 [Step 2](/setup/import#step-2)로 건너뛸 수 있습니다.

파일이 없는 경우 기본 필드와 샘플 .CSV 파일을 다운로드할 수 있습니다.

<Warning>
샘플 .CSV 파일에 포함되어 있는 샘플 커스텀 필드는 (예: Linkedin, Segment, Tools) 샘플일 뿐이라는 점에 유의하세요! 커스텀 필드를 가져오려면 커스텀 필드를 직접 만들어야 합니다.
</Warning>

가져올 .CSV 파일이 준비되면 Step 2로 진행합니다.

### Step 2. 파일 업로드
.CSV 파일을 끌어다 놓거나 가운데를 클릭하여 불러올 .CSV 파일을 선택합니다.

![import2](/images/setup/import2.png)

.CSV 파일이 성공적으로 추가되면 오른쪽 하단의 Next Step을 눌러 계속 진행합니다.

<Tip>
임포트시 디폴트로 첫 번째 행은 Header입니다. 첫 번째 행을 데이터 값으로 처리하려면 왼쪽 상단의 토글 버튼을 전환하여 이 모드를 끕니다.
</Tip>


### Step 3. Match Columns
이 단계에서는 .CSV 파일의 열을 Relate의 데이터 필드와 맵핑할 수 있습니다.

![import1](/images/setup/import.png)

만약 .CSV 파일에 내가 임포트하지 않으려는 필드가 있다면 Skip을 눌러 건너뜁니다.

#### 자동 스마트 퍼지(smart fuzzy) 매칭
Relate은 텍스트를 자동으로 매칭함으로 업로드하는 CSV파일내의 칼럼들과 임포트하려는 필드를 예측해 매칭합니다.

#### 명확한 에러 방지 프로세스 (Fix Validation Error)
수정이 필요한 필드가 있는 경우 가져오기 페이지에서 바로 스프레드시트를 편집하는 것처럼 필드를 편집할 수 있습니다. 로컬 Excel 파일로 돌아갈 필요가 없습니다.

매칭 및 업로드 과정에서 문제가 발생하면 빨간색으로 오류가 표시됩니다.

![Relate Data Import error](/images/setup/import_menu6.png)

모든 열을 일치시키면 다음 단계로 진행합니다.

### Step 4: Import data
`Import Data` 버튼을 클릭하면 확인 메시지가 표시됩니다. 임포트를 확인하면 아래 화면이 표시됩니다.

![successful Relate Data](/images/setup/import_menu7.png)

## 임포트 성공 여부 확인 (Validate successful import)
이제 [워크스페이스](https://app.relate.so/)로 이동하여 가져오기에 성공했는지 확인하세요. 경우에 따라 되돌리기가 필요할 수 있습니다. 자체 되돌리기 기능은 없지만, 앱 왼쪽 하단에 있는 메신저를 통해 팀에 문의해주시면, 빠르게 도와드리겠습니다.

## 자주 묻는 질문 (FAQ)
<Accordion title="Custom Field는 어떻게 임포트하나요?">
  Custom Field를 사용하여 가져오려면, 임포트 하기전에 [Custom Field](https://www.relate.kr/docs/features/custom-fields) 필드를 미리 Relate에 추가해야 합니다.
</Accordion>
