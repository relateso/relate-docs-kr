## Overview
Relate은 별도의 어드민 페이지를 갖고 있습니다. 어드민 페이지에서는 어드민 권한을 가진 사용자만이 변경 사항을 적용할 수 있습니다.

Relate 어드민 페이지 접근 방법:
- 앱에서 워크스페이스를 누르고 [Workspace settings](https://admin.relate.so/)로 이동.
- 브라우저에서 [admin.relate.so](https://admin.relate.so/)로 이동.

## Basics
아래 권한별 탭을 눌러 각 그룹이 (Admin, Members, Guests) 어드민 페이지에서 무엇을 할 수 있는지 확인하세요.
<Tabs>
<Tab title="Admins">
  **Admin 사용자의 권한:**
    - 워크스페이스 이름과 로고를 업데이트합니다.
    - 워크스페이스 기본 통화를 업데이트합니다.
    - Prospect 기능을 활성화/비활성화합니다.
    - 새 멤버를 워크스페이스에 초대합니다.
    - 멤버십을 관리합니다.
    - 이메일을 연결합니다.
    - 다른 소프트웨어 앱과 연동합니다.
    - Zapier에 사용되는 API 키를 관리합니다.
    - Relate로 데이터를 임포트할 수 있습니다.
    - 새 파이프라인 관리 및 생성할 수 있습니다.
    - 파이프라인 단계를 관리할 수 있습니다.
    - 워크스페이스에서 사용되는 커스텀 필드를 관리할 수 있습니다.
</Tab>
<Tab title="Members">
  **Member 사용자의 권한:**
    - 워크스페이스 멤버를 확인할 수 있습니다.
    - 이메일을 연결할 수 있습니다.
    - 다른 소프트웨어 앱과 연동합니다.
    - Zapier에 사용되는 API 키를 관리합니다.
    - Relate로 데이터를 임포트할 수 있습니다.
    - 파이프라인 단계와 파이프라인의 목록을 볼 수 있습니다.
    - 커스텀 필드 목록을 볼 수 있습니다.
</Tab>
<Tab title="Guests">
  **Guest 사용자의 권한:**
    - 워크스페이스 멤버를 볼 수 있습니다.
    - 파이프라인 목록을 볼 수 있습니다.
    - 파이프라인 단계를 볼 수 있습니다.
    - 커스텀 필드 목록을 볼 수 있습니다.
</Tab>
</Tabs>
