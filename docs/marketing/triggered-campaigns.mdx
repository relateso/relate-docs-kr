## Overview

자동 이메일 캠페인(Triggered Campaign)은 연락처(Contact)가 리스트에 추가되면 미리 작성된 메일을 발송하는 기능입니다.

자동 이메일 캠페인(Triggered Campaign)을 활용하면 대량으로 일회성 이메일을 보내는 캠페인(One-off Campaigns)와 달리 **누군가가 폼 응답을 제출하거나, 리스트에 추가되는 등의 이벤트를 트리거(Trigger) 삼아** 자동으로 개인화된 이메일을 보낼 수 있습니다.

예를 들어, 아래와 같이 활용할 수 있습니다.

- **사용자가 폼(Form) 응답을 제출하면** 자동으로 Lists에 추가되고, 미리 작성해둔 이메일이 바로 발송됩니다.
- **고객의 명함을 리멤버 등으로 스캔하면** 바로 리스트(Lists)에 추가한 후, 팔로업 이메일을 발송합니다.
- 미리 작성된 이메일에 이름, 회사와 같은 정보를 개인화하여 넣을 수 있고, 일반 캠페인(One-off Campaigns)과 동일하게 에디터로 쉽게 작성할 수 있습니다.

## 자동 이메일 캠페인 작성 방법

1. 연락처를 추가할 리스트(List)를 만듭니다. 
- 리스트(List)를 만드는 방법은 [이 문서](https://www.relate.kr/docs/features/lists#lists)에서 확인할 수 있습니다.

2. 리스트(List)와 자동 이메일 캠페인(Triggered Campaigns)을 연결합니다.
- 왼쪽 사이드바의 자동 이메일 캠페인(Triggered Campaigns) 클릭 후,  **자동 이메일 캠페인 추가(+ Triggered Campaign)**를 통해 새로운 자동 이메일 캠페인(Triggered campaign) 을 생성합니다.

![triggered-campaigns-kr-01](/images/features/triggered-campaigns/triggered-campaigns-kr-01.png)

- 트리거(Triggers)탭의 **리스트 연결(+ Connect a list)** 클릭 후, 구독 폼(Form)에 연결해둔 리스트(List)를 선택합니다.

![triggered-campaigns-kr-02](/images/features/triggered-campaigns/triggered-campaigns-kr-02.png)

- 리스트(List)에 추가된 시점과 자동 이메일 캠페인을 발송할 시점 사이 간격을 설정할 수 있습니다. (예: 리스트 추가 후 1시간 뒤 이메일 발송)

<Note>
 현재는 특정 리스트에 연락처가 추가된 시점을 트리거로 선택할 수 있으며, 추후 다양한 자동화 옵션이 추가될 예정입니다. 
</Note>

- **중복 발송 허용(Allow duplicate sends) 옵션**을 활성화하여, 여러 개의 리스트(List)에 동일한 연락처가 추가될 때 마다 자동 이메일 캠페인이 발송되도록 설정할 수 있습니다. 

![triggered-campaigns-kr-03](/images/features/triggered-campaigns/triggered-campaigns-kr-03.png)

3. 자동 이메일 캠페인(Triggered Campaigns)을 통해 발송할 이메일을 작성합니다.  
- 자동 이메일 캠페인(Triggered Campaigns)의 작성(Compose) 탭에서 이메일을 작성합니다.  
   - 프리뷰 텍스트, 본문, 푸터까지 모두 작성합니다.  
   - [캠페인 이메일 작성 방법](https://www.relate.kr/docs/marketing/marketing)을 참고하세요.  
   - 우측 상단의 **[Test send]**로 테스트 메일을 보내볼 수 있습니다.

![triggered-campaigns-kr-04](/images/features/triggered-campaigns/triggered-campaigns-kr-04.png)

### 구독 폼(Form)과 연동 방법

구독 폼(Form)과 연동하는 경우, 구독 폼(Form)에 응답을 제출하면 팔로업 이메일이 자동으로 발송됩니다.

1. 구독 폼(Form) 만드는 방법은 [이 문서](https://www.relate.kr/docs/marketing/forms#forms)에서 확인할 수 있습니다.
2. 구독 폼(Form)과 리스트(List)를 연결합니다.
- 왼쪽 탭의 구독 폼(Form) - 원하는 구독 폼(Form) 선택 - 연결된 리스트(Connected Lists) - 리스트 연결(Connect a list)를 눌러 연결하고자 하는 리스트를 선택합니다.

![triggered-campaigns-kr-05](/images/features/triggered-campaigns/triggered-campaigns-kr-05.png)

3. 리스트 연결을 완료하면 구독 폼(Form)에 새로운 응답이 기록될 때 마다 자동으로 선택한 리스트(List)에 연락처(Contact)가 추가됩니다
4. 이후 단계는 위 **자동 이메일 캠페인 작성 방법**을 참고하세요!

### 리멤버와 연동 방법

리멤버와 연동하는 경우, 명함 촬영 시 팔로업 이메일이 자동으로 발송됩니다.

1. 리멤버와 Relate을 연동하는 방법은 [이 문서](https://www.relate.kr/docs/integrations/remember#remember-relate)에서 확인할 수 있습니다.
2. 리멤버와 연동을 완료하면 리멤버를 통해 새로운 연락처(Contact)가 추가될 때마다 자동으로 Relate에서 미리 연결해둔 리스트(List)에 추가됩니다.
3. 이후 단계는 위 **자동 이메일 캠페인 작성 방법**을 참고하세요!


### 참고사항 
- 현재 Contact List에는 중복되는 연락처(이메일)가 추가될 수 없도록 제한되어 있어, 같은 이메일로 중복 발송을 원하신다면 **해당 Contact을 리스트에서 제거 후 다시 추가**하시면 됩니다.
- Triggered Campaign 기능은 유료 요금제(+2k 이상)에서만 사용하실 수 있습니다. 해당 기능은 현재 Spread 에서도 사용하실 수 있으며, Relate 사용자 중 Triggered Campaign 사용을 희망하시는 경우에는 앱 내 우측 하단 문의 채널을 통해 신청해 주시면 별도로 사용 안내 드리겠습니다.
- 앞으로 Triggered Campaigns에 **리스트 기반 자동화 기능**이 추가됩니다! 
   - 자동 이메일 캠페인 외에도, 앞으로 리스트를 중심으로 워크플로우를 자동화하는 기능이 추가될 예정입니다. 
   - 예시: A 리스트에 자동 이메일 캠페인으로 메일 발송 -> 메일 내의 특정 링크를 2회 이상 클릭한 수신자는 B 리스트에 추가 -> B 리스트에 들어있는 연락처를 특정 시퀀스에 수신자에 추가
