# Overview

필터(Filter)와 정렬을 뜻하는 Sort를 통해 Organization, Contact, Deal을 여러가지 조건을 적용해서 내가 확인해야 하는 리스트를 추리거나 정렬할 수 있습니다.

”이번 주 내가 팔로업해야 하는 리드들”과 같은 필터가 좋은 예시입니다. 이 리스트를 추리기 위해서는 아래와 같이 Filter를 만들 수 있습니다.

* `Updated at` + `is after` + 이번 주 월요일

![필터 예시: Updated At is after, First name contains 조건](/images/features/filter-example.png)

### [**​**](http://localhost:3000/features/filter-sorter#filter-sorter)

## 여러가지 Filter/Sorter를 동시 적용하기

1개 이상의 Filter와 Sorter를 적용할 경우, `And` 와 `Or` 중 필요한 조건을 적용할 수 있습니다.

* `And`: Filter A와 Filter B에 모두 해당되는 Organization, People, Deal
* `Or`: Filter A와 Filter B 둘중 하나라도 해당되는 Organization, People, Deal

# 표시 필드 선택 (Display Fields)

![표시 필드(Display fields) 선택 목록](/images/features/display-fields.png)

Organization, People 리스트 오른쪽 상단에 있는 `Display fields` 버튼을 눌러 Organization, People 리스트에서 볼 수 있는 Field들의 순서를 정할 수 있고, 필요하지 않은 필드들을 숨길 수 있습니다.
