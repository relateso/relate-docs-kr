## Overview
Relate에서는 Filter와 Sorter 조건들을 "Views"로 저장하고 팀과 공유할 수 있습니다.

## Basics
View는 Relate내 모든 레코드 타입(Organization, People, Deals)에 대해 적용할 수 있습니다.

## Views 생성하기
View를 생성하기 위해서는 먼저 Filter, Sorter 조건을 선택한 다음, 우측 상단에 있는 `Save as Data view` 버튼을 누릅니다.

View 생성 모달이 보이면, 해당 View 이름을 입력하고 나만 사용할지, 혹은 워크스페이스 유저들 전체와 공유할지 선택합니다 (`Share within this workspace`)

마지막으로 `Create Data view`를 누르면, View가 생성됩니다.

## 많이 사용하는 View 예시
Deal Assignee별 View를 만들어 팀원 별로 담당하는 Deal을 필터해서 볼 수 있습니다.

[Deal Assignee](/features/how-to-manage-deal-assignee)로 View를 만들 경우, 각 영업 담당자가 관리하는 Deal을 확인할 수 있습니다.

특히 파이프라인에 딜 갯수가 많아질 경우, 각 영업 담당자가 관리하는 딜을 확인하기 어렵기 때문에 Data View를 활용하면 좋습니다.

![description-note](/images/features/pipeline-deal-assignee-dataview.png)

위의 스크린샷처럼 특정 담당자가 관리하는 딜을 필터한 상태를 Data View로 저장할 수 있습니다.
