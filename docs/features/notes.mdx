## Overview
각 [Organization](/features/organization/)상세 페이지에서 노트를 만들고, 편집하고, 팀원들과 공유하세요. 템플릿을 만들어 노트의 서식을 자동으로 지정할 수도 있습니다.

## Basics
영업을 개선하는 가장 쉽고 좋은 방법 중 하나는 각 영업 통화 후에 항상 메모를 작성하는 것입니다.

Relate의 노트는 마크다운 구문을 지원하며, 각 초안 상단에 있는 WYSIWYG 스타일 지정 버튼을 사용할 수도 있습니다.

## 노트 생성하기(Creating a note)
노트를 만들려면 고객 상세 페이지 (Organization Detail)로 이동합니다. 그런 다음 `Create note`를 클릭해 새 초안을 엽니다.

Inbox나 Deal 페이지(예: Prospect, Pipeline)에서 각 딜을 열어서 노트를 만들 수도 있습니다.

<Warning>현재 노트들은 드래프트로 저장이 되지 않습니다.</Warning>

## 노트 템플릿(Templates)
자주 사용하는 포맷을 템플릿으로 저장하려면 노트 초안을 연 다음, `Use template`을 클릭합니다.

Relate에는 3가지 디폴트 템플릿이 있습니다.
- Default
- [BANT](https://www.relate.kr/blog/what-is-bant/)
- [MEDDIC](https://www.relate.kr/blog/what-is-meddic/)

`+ Add a new template` 을 클릭해서 새로운 템플릿을 만들 수도 있습니다.
