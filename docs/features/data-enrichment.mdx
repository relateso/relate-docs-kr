## Overview

Relate에서는 Clearbit 연동을 통해 고객 데이터를 자동으로 인리치(enrich)할 수 있습니다. 회사 도메인이나 이메일 주소만 입력해도, 외부 데이터를 수집해서 회사 설명, 직책 등의 세부 정보를 자동으로 고객 데이터에 추가해 주기 때문에 편리하죠. 모든 Relate 사용자는 매월 최대 250개의 데이터를 인리치할 수 있습니다.

### 데이터 인리치먼트(Data Enrichment)는 무엇일까요?

데이터 인리치먼트는 다양한 데이터베이스에서 수집한 정보로 고객 데이터의 내용을 인리치(enrich), 혹은 더 풍부하게 보완할 수 있는 기능입니다. 신뢰할 수 있는 출처에서 가져온 여러 정보를 구조화된 데이터셋에 자동으로 입력해줌으로써 활용하기 좋은 고객 프로필을 생성할 수 있고, 이는 세일즈와 마케팅 활동에 효과적입니다.

## Clearbit 연동하기

1. Relate 앱 왼쪽 상단의 워크스페이스 이름을 클릭하고, **Manage workspace**를 선택해 [Admin](https://admin.relate.so/) 페이지에 접속하세요.
2. **Integrations > Clearbit**으로 이동하세요.
3. `Connect` 버튼을 클릭하고 연동을 시작하세요.
4. 무료 Clearbit API 키가 생성됩니다.
5. Relate 앱 내에서 데이터 인리치먼트를 경험해 보세요.

## Relate에서 데이터 인리치먼트 활용하기

Relate에서 데이터 인리치먼트를 다음과 같이 활용할 수 있습니다.

**신규 레코드**
- 새로운 [조직](https://www.relate.kr/docs/features/organizations)이나 [연락처](https://www.relate.kr/docs/features/people#person)를 추가할 때 회사 도메인 또는 이메일 주소를 입력하면 회사 설명, 직책 등의 세부 정보가 자동으로 추가됩니다.

**기존 레코드**
- 고객 상세 페이지에서 `Enrich` 버튼을 클릭하여 외부 소스에서 추가 데이터를 가져옵니다.

**벌크 인리치먼트**
- 전체 고객 데이터베이스를 인리치하려면, [Relate 팀](https://www.relate.so/forms/ux3pixh54rin5yre)에게 문의하여 커스텀 플랜에 대해 상담을 받아보세요.


<Note>매월 250개의 크레딧 한도를 초과하면, [Clearbit](https://clearbit.com/) 웹사이트에서 추가 크레딧을 구매할 수 있습니다.</Note>
