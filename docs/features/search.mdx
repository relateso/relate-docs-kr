## Overview
Relate에서 데이터를 검색할 수 있는 방법을 소개합니다.

## Basics
Relate에서는 다음의 데이터를 검색할 수 있습니다.

- Organization
- People
- Entry

검색은 모든 화면 상단에 있는 `Search` Bar를 클릭하여 접근할 수 있습니다. 검색창이 나오면, 검색하고자 하는 데이터를 입력하고 엔터를 누르면 검색 결과를 볼 수 있습니다. Entry의 경우 Organization 또는 Contact과 연결되어 있는 Entry (e.g., Deal, Prospect 등)에 대한 검색 결과를 볼 수 있습니다.

## Command + K 로 검색하기
검색창은 모든 화면에서 `Command + K` 단축키를 통해서도 접근할 수 있습니다. 단축키를 누르면 검색창이 나오며, 검색하고자 하는 데이터를 입력하고 엔터를 누르면 검색 결과를 볼 수 있습니다.

<note>
Command + K 기능을 통해 검색뿐만 아니라 노트 생성 및 이메일 작성도 가능합니다.
</note>
