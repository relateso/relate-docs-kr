## Overview
이제 Relate Connect를 통해 링크드인 프로필 데이터 (e.g., 이름, 타이틀, 회사, 이메일 주소 등)를 자동으로 추출하고 Relate 워크스페이스에 자동으로 추가할 수 있습니다.

## How to install

### Requirements
- Google Chrome Browser에서만 사용이 가능합니다.
- 다른 Chromium 기반의 브라우저 (e.g., Arc, Brave, Edge 등) 위에서는 현재 작동하지 않습니다.
- Mac, Linux, Windows OS환경에서 사용이 가능합니다.

### Plasmo Itero 페이지에서 설치

[Plasmo Itero 설치 가이드](https://itero.plasmo.com/ext/ogpinifhnhlfmafbalecnmakopnldfgg)

위 링크를 따라 들어가면 macOS, Linux, Windows 환경에 따라 다른 설치 가이드를 확인할 수 있습니다. 설치 가이드를 따라서 설치를 진행해주세요.

### 익스텐션에서 Relate 로그인하기
익스텐션 설치를 완료한 다음, Browser 상에서 Relate 로그인을 진행합니다. 이미 Relate app에 로그인이 되어 있다면 바로 로그인 처리가 됩니다.

## How it works
데이터를 추출하고자 하는 링크드인 프로필 페이지에서 Relate Connect 익스텐션 버튼을 누르면, 공개된 정보를 기반으로 익스텐션 Form에 데이터를 채우게 됩니다.

![conenct](/images/features/linkedin-connect.png)

Relate에 자동으로 추가하기 위해서는 데이터를 추가할 Workspace를 선택하고, `Add to Relate` 버튼을 누르면 자동으로 데이터를 추가할 수 있습니다.


## 자주 묻는 질문 (FAQs)
<AccordionGroup>
<Accordion title="왜 어떤 프로필에서는 이메일 주소가 보이지 않나요?">
해당 프로필 사용자가 이메일 주소를 비공개 처리한 경우, 이메일 주소가 보이지 않으므로 익스텐션도 이메일 주소를 가져오지 않는 것입니다.

일반적으로 링크드인 1촌인 경우 이메일 주소가 공개되어 있고, 2촌이나 3촌인 경우 비공개처리 되어 있습니다.

이후 추가 익스텐션 업그레이드를 통해 다른 웹상의 공개 정보를 취합하여 이메일을 채우는 기능을 업데이트할 예정입니다.
</Accordion>
