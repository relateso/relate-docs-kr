## Overview
사람(People)은 [Organization](/features/organizations/)에 속하거나 속하지 않을 수 있으며, 고객의 구매 프로세스에 영향을 미치는 모든 이해관계자를 의미합니다. 예를 들어, 고객사 내 담당자, 의사결정권자(챔피언), 구매 담당자 등이 포함될 수 있습니다.


## Concepts

### 피플 그리드(People Grid)
피플 그리드(People Grid)는 워크스페이스에 있는 모든 People을 모아보는 리스트 뷰입니다. 그리드에서 필드를 선택하고 두 번 클릭(또는 `Enter`/`Return`)하여 각 필드를 업데이트할 수 있습니다.

피플 그리드에서 'Open'를 클릭하면 해당 사람과 연결된 고객 상세 페이지가 열립니다.

## Person 추가하기
새로운 Person을 추가하는 방법은 여러 가지가 있습니다.

- People Grid에서 `+Contact` 버튼을 클릭하여 새 사람을 추가합니다.

   ![add-person](/images/features/add-person.png)

- 조직 상세 페이지에서 Contacts 영역의 `+` 버튼을 클릭합니다.

- 새로운 Organization을 추가할 때 모달 하단의 `+Add contact data`를 클릭합니다.

<Tip>
  새로운 Contact를 추가할 때, 조직의 이름을 입력하지 않아도 이메일 주소의 도메인을 기반으로 한 조직 정보가 자동으로 [Organization](/features/organizations)으로 추가됩니다.
</Tip>
