## Overview
사람(People)은 [Organization](/features/organizations/)의 일부입니다. 각 Org의 "Person"은 고객, 챔피언 또는 구매 프로세스에 기여하는 고객측 모든 이해관계자가 될 수 있습니다.

또한 사람은 Organization 없이도 단독으로 존재할 수 있습니다.


## Concepts

### 피플 그리드(People Grid)
피플 그리드(People Grid)는 워크스페이스에 있는 모든 People을 모아보는 리스트 뷰입니다. 그리드에서 필드를 선택하고 두 번 클릭(또는 `Enter`/`Return`)하여 각 필드를 업데이트할 수 있습니다.

피플 그리드에서 'Open'를 클릭하면 해당 사람과 연결된 고객 상세 페이지가 열립니다.

## Person 추가하기

피플 그리드에서 `+Contact` 버튼을 클릭하여 새 사람을 추가합니다.

![add-person](/images/features/add-person.png)

사람을 추가하는 다른 방법도 있습니다.

1. 고객 상세 페이지로 이동하여 사람 탭 아래의 `+` 버튼을 클릭합니다.

2. 새로운 Organization을 만들 때 `+Add contact data`를 클릭합니다.
