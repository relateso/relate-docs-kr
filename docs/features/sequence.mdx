## Overview

이제 Relate Sequence를 사용하여 며칠에 걸쳐 여러 이메일 자동으로 보낼 수 있습니다. Relate Sequence 베타는 B2B 스타트업을 위해 만들어진 간단한 이메일 자동화 툴로 현재 모든 Relate 유료 사용자가 베타 버전으로 사용할 수 있습니다.

## 베타 업데이트

Relate Sequence 베타 버전은 아직 대량의 콜드 이메일을 보내는 목적이 아닌 소수의 고객에게 자동으로 팔로업 이메일을 보내는 목적으로 사용해 주세요.  예를 들어 다음의 경우에 Relate Sequence 베타 기능을 사용하면 좋습니다. 

- 딜이 진행 중인 고객에게 보내는 팔로업 이메일. 
- 인바운드 문의를 한 잠재 고객과 일정을 잡기 위한 팔로업 이메일.

## Relate Sequence 사용하는 법

![relate-sequence-menu](/images/features/relate-sequence-menu.png)

Relate Sequence 베타를 사용하기 위해서는 Relate 메뉴 왼쪽에 있는 Sequnces (beta) 탭을 클릭합니다.

그 이후, 시퀀스를 추가하고, 템플릿과 Mail Merge 기능을 활용해서 보낼 이메일을 추가한 뒤, 이메일을 보낼 담당자를 추가하면 됩니다. 

### Variables 사용하기

![mail-merge](/images/features/mail-merge.png)

Variables('mail merge'이라고도 함)를 사용하면 시퀀스에서 보낼 이메일이나, 직접 수동으로 보내는 이메일을 전보다 쉽고 빠르게 보낼 수 있습니다. 현재 지원하는 Variables는 다음과 같습니다:

- {{full_name}}: 이름 (이후 버전에서는 성, 이름을 나눌 예정입니다.)
- {{title}}: 타이틀, 직책
- {{email}}: 담당자 이메일
- {{phone}}: 담당자 전화번호
- {{organization}}: 회사명
- {{domain}}: 회사의 URL 주소
- {{sender_name}}: 보내는 사람 이름
- {{sender_email}}: 보내는 사람 이메일
- {{now_day}}: 오늘 날짜 (예: 11/15일인 경우 "15")
- {{now_month}}: 이번 달 (예: November)
- {{now_year}}: 올해 연도 (예: 2023)
- {{now_weekday}}: 요일 (예: Wednesday)

## 이후 업데이트 계획

다음의 기능이 이후 업데이트 버전에서 추가될 예정입니다.

- Organization이나 People 페이지에서 다양한 필터를 적용하여 리스트를 만든 후, 쉽게 시퀀스에 추가할 수 있습니다. 
- 이메일을 보낼 때 다른 이메일 주소로 보내도록 선택할 수 있습니다. 
- Open rates, click rates, bounce rates를 확인할 수 있습니다.
- Sequence에서 보낼 예정인 이메일에 대한 Preview를 제공합니다.
