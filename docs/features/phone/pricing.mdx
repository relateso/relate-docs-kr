## 요금제

Call to Follow-up 요금은 **① 연동 방식**과 **② 통화량**에 따라 결정됩니다.

## 1. 연동 방식

두 가지 방식이 있으며, 방식에 따라 셋업 비용이 달라집니다.

- **CTI (Relate 앱 내 인터페이스)** — Relate 화면 안에서 바로 통화합니다. **고객 기록을 함께 보면서 통화**할 수 있고, 전화를 걸거나 받을 때 **해당 연락처 페이지로 자동 이동**합니다.
- **소프트폰 (Linphone 등)** — Linphone 같은 외부 소프트폰 앱으로 통화합니다. **가볍게 시작해보고 싶을 때** 적합합니다.

## 2. 사용 요금 (통화량 기준)

**팀 전체 번호 연동을 $250/월부터 시작할 수 있습니다.**

<Note>
  **전화 통화·통화 녹음·AI 요약이 모두 포함된 가격**입니다. 통신 요금이나 녹취(STT)·AI 처리 비용이 **따로 붙지 않습니다.**
</Note>

직접 구성하면 통신사·녹음·AI를 각각 계약해야 하지만, Call to Follow-up은 하나의 가격으로 끝납니다.

| | 직접 구성 (따로따로) | Call to Follow-up |
| --- | --- | --- |
| 전화 회선·통신 | 통신사 별도 계약 | **포함** |
| 통화 녹음 | 별도 솔루션 | **포함** |
| 녹취(STT)·AI 요약 | STT·AI 비용 별도 | **포함** |
| CRM 연동 | 직접 연동·개발 | **기본 내장** |
| 청구 | 여러 곳에 각각 | **하나의 가격** |

- **$250 / 월** — 팀 전체 번호 연동, **20시간 통화량 포함**, 이후 **사용량 기반**으로 과금됩니다.
- **셋업 비용 별도** — Relate 내 전화 인터페이스(CTI) 연동 방식인지, 외부 소프트폰 사용 방식인지에 따라 셋업 비용이 다릅니다.

<Note>
  연동 방식과 예상 통화량에 따라 요금·셋업 비용이 달라집니다. 상세한 내용이 궁금하시면 [베타테스트 신청](https://public.relate.so/p/forms/7i5k9xq20jzttzk9?utm_source=guidedoc)으로 문의해 주세요.
</Note>

<Warning>
  Call to Follow-up은 **비공개 베타**로, 요금 체계는 정식 출시 전까지 **변경될 수 있습니다.**
</Warning>
