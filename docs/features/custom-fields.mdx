## Custom Field

![Add Contact Custom Field](/images/features/add-contact-custom-field.png)

위 스크린샷 처럼 Organization (Lead), People (Contact), Deal 페이지에서 다양한 형식의 custom field들을 추가할 수 있습니다.

Custom Field 설정은 Relate 앱내에서 워크스페이스 일립시스(⋅⋅⋅)를 누르면 나오는 메뉴에서 `Custom fields` 에 들어가서 하실 수 있습니다.

![Custom fields](/images/features/custom-field.png)

Custom Field 설정 페이지 진입은 브라우저에서 세팅 페이지로 바로 가셔도 보입니다.

![Manage Custom Fields in Company](/images/features/manage-custom-fields-in-company.png)

## Custom Field 종류

| Field Data Type    | Note                                                                                                                                                                              |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Text               | 일반 텍스트 형식을 지원합니다. 한 줄짜리 짧은 설명을 남기기에 좋습니다.                                                                                                           |
| Multi-line text    | 한 줄 이상의 텍스트 형식을 지원합니다. 긴 설명을 남기기에 적합합니다.                                                                                                             |
| Number             | 숫자 형식입니다.                                                                                                                                                                  |
| Date               | 날짜를 표시할 수 있습니다.                                                                                                                                                        |
| Datetime           | 날짜와 시간을 표시할 수 있습니다.                                                                                                                                                 |
| Select             | 데이터 입력시 드롭다운에서 선택할 옵션을 입력할 수 있습니다. 데이터는 쉼표(,)로 구분합니다.<br /><br />1개 이상의 옵션 선택을 원할시에는 `Accept multiple values`에 체크해주세요. |
| Resource: Users    | 우리 워크스페이스의 유저를 연결할 수 있습니다. 담당자를 지정하는데 적합합니다.                                                                                                    |
| Resource: Contacts | 해당 고객사에 소속되어 있는 컨택을 추가할 수 있습니다.                                                                                                                            |

<Note>
  Custom Field를 잘 활용하면, 우리 회사가 관리하고자 하는 다양한 고객의 정보를
  수집하고 관리할 수 있게 됩니다. 이후 Deal, URL이나 이메일 주소와 같은 Field도
  추가될 예정입니다.
</Note>

<Note>
  Custom Field 수정 사항을 바로 반영해서 보려면 Relate 앱을 새로 실행하거나,
  이미 실행 중이라면 Ctrl+Shift+R (macOS: Cmd+Shift+R)를 눌러 새로고침을 하면
  됩니다.
</Note>
