Relate 가이드 문서의 주요 추가·개선 내역입니다. 제품 기능 전체 변경 이력은 [Changelog](https://www.relate.kr/changelog/)에서 확인하실 수 있습니다.

<Update label="2026-06-29" tags={["기능"]}>
  ## 예약 링크 정보 미리 채우기(Prefill) 안내 추가

  [미팅 스케줄러](/features/calendar) 문서에 예약 링크 뒤에 쿼리 파라미터를 붙여 이메일·이름 등을 미리 채우는 방법을 추가했습니다. 기본 제공 필드와 Advanced 커스텀 필드(identifier) 모두 지원하며, 미리 채운 값을 수정 불가(읽기 전용)로 잠그는 옵션도 함께 안내합니다.
</Update>

<Update label="2026-06-21" tags={["기능", "Beta"]}>
  ## Call to Follow-up (Beta) 문서 추가

  통화하면 Relate이 자동으로 기록·요약하고 후속 액션까지 이어주는 **[Call to Follow-up](/features/phone/overview)** 가이드를 신설했습니다. 개요·시작하기·요금제·자주 묻는 질문으로 구성했으며, 현재 비공개 베타로 신청을 받습니다.
</Update>

<Update label="2026-06-18" tags={["연동"]}>
  ## 구글 컨택 · 리멤버 연동 문서 추가

  구글 계정을 연결해 구글 컨택 연락처를 Relate로 자동으로 불러오는 [구글 컨택 · 리멤버에서 가져오기](/setup/google-contacts) 문서를 추가했습니다. 리멤버 명함 연락처도 Zapier 없이 가져올 수 있으며, '연락처 가져오기' 섹션을 신설했습니다.
</Update>

<Update label="2026-06-16" tags={["API"]}>
  ## Relate Public API 문서 추가

  API 섹션을 신설하고 [API 레퍼런스](/api/introduction)를 추가했습니다. 인증·페이지네이션부터 Contacts·Organizations·Notes·Custom Fields·Lists/Processes·Entries·SMS·Calendar·Recordings까지 다룹니다.
</Update>

<Update label="2026-06-15" tags={["AI"]}>
  ## Relate MCP 가이드 추가

  AI 섹션을 신설하고 [Relate MCP](/ai/relate-mcp) 문서를 추가했습니다. Claude Code·Codex 등 AI 에이전트를 Relate에 연결하는 방법, MCP로 할 수 있는 작업, 활용 예시를 담았습니다.
</Update>

<Update label="2026-04-03" tags={["기능"]}>
  ## 노트테이커(Notetaker) 문서 업데이트

  미팅 노트테이커 관련 가이드를 업데이트했습니다.
</Update>

<Update label="2026-02-05" tags={["기능"]}>
  ## 미팅(Meetings) 문서 개선

</Update>

<Update label="2026-02-03" tags={["기능"]}>

  Pulse, Forms 관련 가이드를 추가·업데이트했습니다.
</Update>

</Update>
