## Overview

명함 저장 서비스인 Remember에 저장한 명함 연락처를 Relate로 자동으로 가져올 수 있습니다. 리멤버 앱/웹 모두 가능합니다.

이제 **Zapier 같은 별도 자동화 도구 없이**, 리멤버를 구글 주소록(구글 컨택)과 연동하고 Relate에서 구글 컨택을 연결하면 리멤버 명함 연락처가 자동으로 들어옵니다.

연동은 두 단계로 진행됩니다.

1. **리멤버 → 구글 주소록 자동 저장 설정** — 리멤버 명함이 구글 컨택에 자동으로 쌓이도록 설정합니다.
2. **Relate에서 구글 컨택 연동** — 구글 컨택을 통해 리멤버 연락처가 Relate로 들어옵니다.

## 1. 리멤버 - 구글 주소록(구글 컨택)에 자동 저장

### 웹(PC)에서 연동하기

![remember 1](/images/integrations/remember-1.png)

자주 사용하는 소셜 계정 혹은 이메일로 회원가입 후 로그인 해주세요.

![remember 2](/images/integrations/remember-2.png)

우측 상단 프로필을 누르고, **설정-구글 주소록 자동저장**을 선택해 계정을 연동해주세요.

- 연동하는 계정은 로그인한 계정과 달라도 괜찮습니다.
- 이 기능을 사용하시면 리멤버에 저장한 명함 정보가 구글 연락처에 자동 추가되며, 명함 정보를 수정할 때에도 연락처가 자동 갱신됩니다.

![remember 3](/images/integrations/remember-3.png)

원하는 부가 옵션을 설정한 뒤 확인을 누르면 연동이 완료됩니다.
연락처에 저장되는 이름은 `이름만` 설정해두어야 Relate Contact에 이름이 들어가니 유의해주세요.

### 앱(모바일)에서 연동하기

자주 사용하는 소셜 계정 혹은 이메일로 회원가입 후 로그인 해주세요.

![remember 6](/images/integrations/remember-6.png)
![remember 7](/images/integrations/remember-7.png)

메인 화면의 오른쪽 상단 **더보기**를 누르고, **설정**을 선택합니다.

![remember 8](/images/integrations/remember-8.png)

**휴대폰 연락처에 저장**을 누릅니다.

![remember 9](/images/integrations/remember-9.png)

**구글 주소록**을 누른 후, **구글 주소록에 자동 저장**을 켭니다.

![remember 11](/images/integrations/remember-11.png)
![remember 12](/images/integrations/remember-12.png)

연동된 연락처 계정을 확인하고, **저장 옵션 설정**을 눌러 연동 설정을 완료합니다.

## 2. Relate에서 구글 컨택 연동

리멤버 명함이 구글 컨택에 자동 저장되도록 설정했다면, 이제 Relate에서 구글 컨택을 연결하면 됩니다. 구글 컨택을 통해 리멤버 명함 연락처가 Relate로 자동으로 들어옵니다.

설정 방법은 [구글 컨택 · 리멤버에서 가져오기](/setup/google-contacts)를 참고하세요.

<Note>
  구글 컨택의 `Remember` 그룹에 속한 연락처는 Relate에서 **리멤버 연락처로 분류**되어 저장됩니다. 즉, 구글 컨택 하나만 연결하면 리멤버 명함 연락처까지 함께 따라옵니다.
</Note>

<Warning>
  **구글 계정이 동일한지 확인하세요.**

  리멤버에서 명함을 자동 저장하도록 연동한 **구글 계정**과, Relate에 연결한 **구글 계정**이 서로 **동일해야** 리멤버 연락처가 정상적으로 들어옵니다. 두 계정이 다르면 리멤버 명함이 Relate로 동기화되지 않으니 꼭 확인해 주세요.
</Warning>
